# Whatbookis - TFG (Desarrollo de Aplicaciones Web) 🎓

## Descripción 📚

**Whatbookis** es una aplicación web interactiva en la que los usuarios deben adivinar el libro basándose únicamente en su primer párrafo. El juego desafía a los jugadores a combinar su memoria y apreciación literaria para identificar libros famosos. 

Este proyecto forma parte de mi Trabajo de Fin de Grado (TFG) en Desarrollo de Aplicaciones Web (DAW). He decidido realizar este proyecto como una oportunidad para aprender y poner en práctica nuevas tecnologías y herramientas, como **React**, **Tailwind CSS**, y otras librerías populares.


## Tecnologías utilizadas 🛠️

En este proyecto he utilizado las siguientes tecnologías:

- **Frameworks**:
  - **React**: Utilizado para el desarrollo de la interfaz de usuario (UI) y la gestión de estados mediante hooks (`useState`, `useEffect`, etc.).

- **CSS Frameworks**:
  - **Tailwind CSS**: Para la creación de una interfaz de usuario moderna y adaptativa. Tailwind CSS me ha permitido desarrollar rápidamente una interfaz flexible y personalizable.

- **Routing**:
  - **React Router**: Para gestionar la navegación entre diferentes páginas del proyecto.

- **Superset**:
  - **TypeScript**: Para escribir un código más seguro y escalable, aprovechando los beneficios del tipado estático.

- **API**:
  - **Fetch API**: Para obtener datos externos, en particular los libros, utilizando la API pública de **Gutendex**.


## Funcionalidades ⚙️

- **Pantalla principal**: Los usuarios pueden ver una lista de libros con su primer párrafo, y tienen que adivinar el título del libro.
- **Página del quiz**: Los usuarios pueden jugar y adivinar el título del libro basándose en el primer párrafo proporcionado por la aplicación.
- **Respuestas y feedback**: Después de cada intento, la aplicación proporciona feedback al usuario sobre si la respuesta fue correcta o incorrecta.
